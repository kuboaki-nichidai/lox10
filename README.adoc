== Lox文法の階層化 -- 6.1

演算の優先度（precedence）:: どの演算子を先に評価するかを決める。
演算の結合度（assoiativity）:: 同じ優先度の演算子を並べたときに、どの演算子を優先するかを決める。


Loxでは、Cと同様の優先順位を採用する。

=== 文法の階層化

Cと同様の優先順位を採用するために、文法を階層化する。


.文法の階層化（下に行くほど優先度が高い）
[source]
----
espression  ::= ...  /* 式の規則 */
equality    ::= ...  /* 等価の規則 */
comparison  ::= ...  /* 比較の規則 */
term        ::= ...  /* 項の規則 */
factor      ::= ...  /* 因子の規則 */
unary       ::= ...  /* 単項式の規則 */
primary     ::= ...  /* 一次式の規則 */
----
